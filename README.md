Este projeto demonstra a construção de um pipeline de análise de dados em nuvem utilizando serviços da AWS. A solução utiliza o Amazon S3 para armazenamento dos dados, o AWS Glue para processamento e organização das informações e o Amazon QuickSight para análise e visualização por meio de dashboards interativos.

O projeto tem caráter educacional e demonstrativo, aplicando conceitos de Cloud Computing, ETL e Business Intelligence, e pode ser utilizado como base para estudos, trabalhos acadêmicos e portfólio profissional.
